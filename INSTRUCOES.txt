# Instalação e execução do projeto

1. Instale o Python 3.14 (ou superior) no Windows.
2. Instale o PostgreSQL (versão 17 recomendada).
3. Abra o terminal na pasta do projeto.
4. Execute o instalador:
   python install.py
   - Isso instalará todas as dependências e criará/atualizará o banco de dados.
5. Para rodar o servidor:
   python app.py
6. Acesse no navegador:
   http://127.0.0.1:5000
